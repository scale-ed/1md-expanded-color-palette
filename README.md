# 1MD Expanded Color Palette

Exploratory palette options for extending 1MD Nutrition's brand system into
high-saturation social ads on Meta — a compromise between media buyers (who
want highly saturated, scroll-stopping colors) and Brand (who want a fixed,
reusable palette instead of one-off color choices per ad).

**[View the presentation →](https://scale-ed.github.io/1md-expanded-color-palette/)**

## Files

- `expanded-palette-presentation.html` — stakeholder-ready presentation (six palette directions)
- `expanded-palette-options.html` — v1 working file
- `expanded-palette-options-v2.html` — v2 working file (revision notes/rationale included)
- `Assets/` — reference screenshot of the current 1MD Figma color system
